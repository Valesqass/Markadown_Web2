# MDtic

MDtic est un éditeur de fichiers markdown réalisé avec React.js, proposant une interface au style rétro.

## Installation

Utilisez le package manager [npm](https://docs.npmjs.com/about-npm) pour installer MDtic.

```bash
npm i
```

## Utilisation

```bash
npm run dev
```
